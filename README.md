# Professional Todo App

A modern and responsive Todo App built using HTML, CSS, and JavaScript with full CRUD operations and LocalStorage support.

---

## Features

- Add Todo
- Display Todos
- Edit Todo
- Delete Todo
- LocalStorage Support
- Responsive Design
- Modern Google Style UI
- Input Validation
- Hover Effects

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

---

## CRUD Operations

### Create
Users can add new todos with a date.

### Read
Todos are dynamically displayed on the screen.

### Update
Users can edit existing todos.

### Delete
Users can remove todos permanently.

---

## LocalStorage

Todos are stored in the browser using LocalStorage.

```javascript
localStorage.setItem('todos', JSON.stringify(todos));
```

Data is loaded using:

```javascript
JSON.parse(localStorage.getItem('todos'));
```
